# penny

> Pay only when it answers. Premium AI chat without the monthly subscription.

Pay-per-message AI billed in cUSD on the Celo network. Top up once, send messages, settle on chain.

## Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- TypeScript

## Develop

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000.
